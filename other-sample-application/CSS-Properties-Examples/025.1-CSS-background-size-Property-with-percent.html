CSS background-size Property

Specify the size of a background image with percent:

background-size: 100% 100%:

Hello World

The size of the background image is set in percent of the parent element; 100% width and 100% height.

background-size: 75% 50%:

Hello World

The size of the background image is set in percent of the parent element; 75% width and 50% height.

Note: The background-size property is not supported in Internet Explorer 8 and earlier versions.
