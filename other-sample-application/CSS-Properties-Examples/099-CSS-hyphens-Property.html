CSS hyphens Property

Set different hyphenations:

The hyphens Property

hyphens: none

No hyphen; overflow if needed.

A veryvery­very long word.

hyphens: manual

Hyphen only at ‐ or ­ (if needed).

A veryvery­very long word.

hyphens: auto

Hyphen where the algorithm is deciding (if needed).

A veryvery­very long word.
