CSS animation-timing-function Property

To better understand the different timing function values : speed curves are defined with the cubic-bezier function:

linear
cubic-bezier(0,0,1,1);
ease
cubic-bezier(0.25,0.1,0.25,1);
ease-in
cubic-bezier(0.42,0,1,1);
ease-out
cubic-bezier(0,0,0.58,1);
ease-in-out
cubic-bezier(0.42,0,0.58,1);

Note: The animation-timing-function property is not supported in Internet Explorer 9 and earlier versions.
