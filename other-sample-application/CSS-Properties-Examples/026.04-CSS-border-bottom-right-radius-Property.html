CSS border-bottom-right-radius Property

Few Examples

Add rounded borders to the bottom-right corner of two <div> elements:

border-bottom-right-radius: 25px:

The border-bottom-right-radius property defines the radius of the bottom-right corner.

border-bottom-right-radius: 50px 20px:

If two values are set; the first one is for the bottom border, the second one for the right border.

Add rounded borders to the bottom-right corner in percent:

border-bottom-right-radius: 40%:

Here, the border-bottom-right-radius property is set to 40%. Also notice that the background is clipped at the border.

CSS Syntax

border-bottom-right-radius: length|% [length|%]|initial|inherit;

Note: If you set two values, the first one is for the bottom border, and the second one for the right border. If the second value is omitted, it is copied from the first. If either length is zero, the corner is square, not rounded.
