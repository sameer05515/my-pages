CSS border-bottom-left-radius Property

Few Examples

Add rounded borders to the bottom-left corner of two <div> elements:

border-bottom-left-radius: 25px:

The border-bottom-left-radius property defines the radius of the bottom-left corner.

border-bottom-left-radius: 50px 20px:

If two values are set; the first one is for the bottom border, the second one for the left border.

Add rounded borders to the bottom-left corner in percent:

border-bottom-left-radius: 40%:

Here, the border-bottom-left-radius property is set to 40%. Also notice that the background is clipped at the border.
